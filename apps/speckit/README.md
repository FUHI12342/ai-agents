# Speckit

A minimal Python CLI application for processing input and generating output.

## Features

- CLI interface for input processing
- File generation capabilities
- Pytest-based testing

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python src/main.py --input "hello" --output output.txt
```

## Development

Run tests:
```bash
./scripts/test.ps1
```

Release backup:
```bash
./scripts/release_backup.ps1
