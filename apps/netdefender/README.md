# NetDefender

A network defense and audit tool for analyzing own network logs, snapshots, and detecting differences. This tool is strictly for internal use and does not perform external scanning or tracking.

## Features

- Log analysis and diff detection
- Snapshot comparison
- CLI interface for easy automation

## Installation

1. Ensure Python 3.8+ is installed
2. Install dependencies: `pip install -r requirements.txt`
3. Run tests: `./scripts/test.ps1`

## Usage

```bash
python src/main.py --help
```

## Development

- Run tests: `./scripts/test.ps1`
- Create backup: `./scripts/release_backup.ps1`
